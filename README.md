# Net audio stream
Vibe coded simple mic streaming app to send audio over network

> [!WARNING]  
> This project was vibe-coded! (and I dont approve vibe-coding)

## Usage

```bash
cargo run -- --mode devices
```

Shows all available input and output devices and modes

```bash
cargo run -- --mode server
```

Runs app in server mode

```bash
cargo run -- --mode client
```

Runs app in client mode
